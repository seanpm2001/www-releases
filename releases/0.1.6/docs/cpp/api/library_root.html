C++ API Reference

The following sections present the C++ API Reference. All types are within the sixtyfps namespace and are accessible by including the sixtyfps.h header file.

If you choose to load .60 files dynamically at run-time, then you can use the classes in sixtyfps::interpreter, starting at sixtyfps::interpreter::ComponentCompiler. You need to include the sixtyfps_interpreter.h header file.
