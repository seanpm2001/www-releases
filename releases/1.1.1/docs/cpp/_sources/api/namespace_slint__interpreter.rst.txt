Namespace slint::interpreter
============================


Detailed Description
--------------------

The types in this namespace allow you to load a .slint file at runtime and show its UI.

You only need to use them if you do not want to use pre-compiled .slint code, which is the normal way to use Slint.

The entry point for this namespace is the :ref:`exhale_class_classslint_1_1interpreter_1_1ComponentCompiler`, which you can use to create :ref:`exhale_class_classslint_1_1interpreter_1_1ComponentDefinition` instances with the ComponentCompiler::build_from_source() or ComponentCompiler::build_from_path() functions.
