C++ API Reference#

The following sections present the C++ API Reference. All types are within the slint namespace and are accessible by including the slint.h header file.

If you choose to load .slint files dynamically at run-time, then you can use the classes in slint::interpreter, starting at slint::interpreter::ComponentCompiler. You need to include the slint_interpreter.h header file.
