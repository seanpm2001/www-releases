Namespace sixtyfps::interpreter
===============================


.. contents:: Contents
   :local:
   :backlinks: none




Detailed Description
--------------------

The types in this namespace allow you to load a .60 file at runtime and show its UI.

You only need to use them if you do not want to use pre-compiled .60 code, which is the normal way to use SixtyFPS.

The entry point for this namespace is the :ref:`exhale_class_classsixtyfps_1_1interpreter_1_1ComponentCompiler`, which you can use to create :ref:`exhale_class_classsixtyfps_1_1interpreter_1_1ComponentDefinition` instances with the ComponentCompiler::build_from_source() or ComponentCompiler::build_from_path() functions.
